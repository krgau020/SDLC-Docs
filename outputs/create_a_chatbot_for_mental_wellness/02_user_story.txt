## Mental Wellness Chatbot User Stories

**1. As a User, I want to be able to initiate a conversation with the chatbot, so that I can start exploring my feelings and experiences.**

*   **Acceptance Criteria:**
    *   The chatbot should have a clear and welcoming initial message.
    *   The chatbot should respond to simple greetings like "Hello," "Hi," or "Good morning."
    *   The chatbot should provide instructions on how to interact (e.g., "Just type your thoughts and feelings.").

**2. As a User, I want to be able to express my feelings and thoughts in a free-text format, so that I can process my emotions.**

*   **Acceptance Criteria:**
    *   The chatbot should accept free-text input from the user.
    *   The chatbot should not censor or judge the user's input (within ethical boundaries – see note below).
    *   The chatbot should display the user’s input clearly.

**3. As a User, I want the chatbot to offer a range of guided exercises (e.g., breathing exercises, mindfulness prompts), so that I can learn coping mechanisms.**

*   **Acceptance Criteria:**
    *   The chatbot should offer at least 3 different types of guided exercises.
    *   Each exercise should have clear instructions.
    *   The chatbot should provide an option to repeat an exercise.

**4. As a User, I want the chatbot to provide information about common mental health conditions (e.g., anxiety, depression), so that I can better understand my experiences.**

*   **Acceptance Criteria:**
    *   The chatbot should provide brief, factual information about at least 3 common mental health conditions.
    *   The information should be sourced from reputable sources (e.g., WHO, NHS).
    *   The chatbot should clearly state that it is not a substitute for professional medical advice.

**5. As a User, I want to be able to set a mood or intention for the conversation, so that the chatbot can tailor its responses.**

*   **Acceptance Criteria:**
    *   The chatbot should allow the user to select from a predefined list of moods (e.g., “Happy,” “Sad,” “Anxious,” “Stressed”).
    *   The chatbot should acknowledge the selected mood.
    *   The chatbot’s responses should be relevant to the selected mood (to a reasonable extent).

**6. As a User, I want the chatbot to offer resources for professional help (e.g., helpline numbers, websites), so that I can seek support if needed.**

*   **Acceptance Criteria:**
    *   The chatbot should provide at least 3 relevant resources.
    *   The resources should include contact information and website URLs.
    *   The chatbot should clearly state that it is not a substitute for professional help.

**7. As a User, I want the chatbot to provide a way to end the conversation, so that I can leave when I’m finished.**

*   **Acceptance Criteria:**
    *   The chatbot should offer a clear option to end the conversation (e.g., "End Chat," "Goodbye").
    *   Ending the conversation should gracefully terminate the interaction.
